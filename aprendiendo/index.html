      </div>

Modules


Un módulo define una aplicación.

Un módulo se declara de la siguiente manera: 







      angular.module('Nombre_del_modulo', ['ngRoute']);

Lo enlazamos a la vista en la cual trabajaremos con la directiva: 






      <html lang="es" ng-app="Nombre_del_modulo">
Directives


Sirven para encapsular código HTML
Definimos una directiva de la siguiente manera:







      angular.module('myDirectives', []).
          directive('myAwesomeDirective', function() {});

Usandolo en HTML





      <my-awesome-directive> </my-awesome-directive>
    
 ng-app, ng-init, ng-model, ng-repeat
Models


Permite obtener la información ingresada por el usuario en
algún elemento del formulario







    <div ng-app="myApp" ng-controller="myCtrl">
        Name: <input ng-model="name">
        <h1>You entered: {{name}}</h1>
    </div>

    <script>
        var app = angular.module('myApp', []);
        app.controller('myCtrl', function($scope) {
            $scope.name = "John Doe";
        });
    </script>
Controllers


Permite obtener la información ingresada por el usuario en
algún elemento del formulario







      <div>
        Full Name: {{firstName + " " + lastName}}
      <div/>
      <script>
        var app = angular.module('myApp', []);
        app.controller('myCtrl', function($scope) {
          $scope.firstName = "John";
          $scope.lastName = "Doe";
        });
      <script/>
    
 Resultado: Full Name: John Doe
Scopes


Hace visible la información necesaria para implementar la aplicación,
desde el controlador a la vista y desde la vista al controlador. 








      var app = angular.module('myApp', []);
      app.controller('myCtrl', function($scope) {
        $scope.carname = "Volvo";
      });

    
 Resultado: Volvo
Filters


Sirven para transformar los datos o dar formato a los datos.



  • currency Dar formato a un número a un formato de moneda.
  • date Dar formato a una fecha a un formato especificado.
  • filter Seleccionar un subconjunto de elementos de una arreglo.
  • limitTo Limita un arreglo o cadena.
  • lowercase Dar formato a una cadena a minúsculas.
  • number Dar formato a un número en una cadena.
  • orderBy Ordena un arreglo por una expresión.
  • uppercase Dar formato a una cadena a mayúsculas.
















      <p>The name is {{ lastName | uppercase }}</p>

    
 Resultado: The name is DOE
Events


Nos permiten usar funciones AngularJS en ciertos eventos de usuarios.




  • ng-mousedown.
  • ng-mouseup.
  • ng-click.
  • ng-mouseleave.
  • ng-mouseover.









    <div ng-app="myApp" ng-controller="myCtrl">
    <h1 ng-mousemove="count = count + 1">Over me!</h1>
    <h2>{{ count }}</h2>
    </div>
          <script>
          var app = angular.module('myApp', []);
            app.controller('myCtrl', function($scope) {
              $scope.count = 0;
          });</script>

Validations


AngularJS ofrece validaciones desde el lado del cliente
al monitorear el estado de los formualrios y campos a llenar










      <body ng-app="">
          <form name="myForm">
          <input type="text" name= "myInput"
          ng-model="myInput">
          </form>
          <p>The input's valid state is:</p>
          <h1>{{myForm.myInput.$valid}}>/h1>
      <body/>
